üåê Step 2: Custom Domain Setup for ANNSATHER.COM

Option A: Keep with Network Solutions (Recommended)

Pros: No transfer fees, faster setup, no risk of downtime
Cons: Manage domain and hosting in separate locations
Steps:
  1. In Amplify Console ‚Üí Domain Management ‚Üí "Add domain"
  2. Enter annsather.com
  3. Amplify provides DNS records to add
  4. Log into networksolutions.com with domain credentials
  5. Navigate to DNS Management for annsather.com
  6. Add the CNAME records provided by Amplify:
    • CNAME record for SSL verification
    • CNAME record for www subdomain
  7. Wait for DNS propagation (15 minutes - 2 hours)

Option B: Transfer to AWS Route 53

Pros: Centralized AWS management, programmatic DNS control
Cons: Transfer fee (~$12), 60-day lock period, more complex setup
Required Actions Before Transfer:
1. Contact Network Solutions to disable clientTransferProhibited
2. Request authorization code (EPP code)
3. Ensure contact information is current
Transfer Steps:
  1. Prepare at Network Solutions:
    • Log into networksolutions.com
    • Disable domain transfer lock
    • Request authorization code
    • Backup current DNS settings
  2. Initiate Transfer in Route 53:
    • Go to Route 53 Console ‚Üí "Transfer domain"
    • Enter annsather.com and authorization code
    • Provide contact information
    • Confirm transfer fee payment (~$12)
  3. Complete Transfer Process:
    • Respond to confirmation emails
    • Transfer typically takes 5-7 days
    • Once complete, configure Amplify domain

  • Contact AWS Support if persistent
